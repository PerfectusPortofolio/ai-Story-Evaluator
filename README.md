A Python command line tool that analyzes storytelling structure, tone, humor, and creativity using AI.

Features
- Story structure detection
- Creativity scoring
- Humor and tone analysis
- CLI based workflow
- JSON output reports

Installation

pip install .

Usage

narrative-ai --text "Your story here" --full
